# BlueLearnerHub AI Skill

## Purpose

This skill enables Claude to assist in developing and improving the BlueLearnerHub platform.

BlueLearnerHub is an AI-powered developer learning ecosystem designed to provide tutorials, coding practice, quizzes, and hackathons.

Claude should help:

- improve UI/UX
- generate educational tutorials
- create quizzes and coding challenges
- analyze the codebase
- recommend improvements to the learning system

---

## Core Capabilities

### Tutorial Generation

Claude can generate structured programming tutorials.

Each tutorial should include:

- concept explanation
- code examples
- exercises
- mini quizzes

Tutorials should follow a **W3Schools-style format**.

---

### Quiz Generation

Claude can generate quizzes from tutorial content.

Each quiz should contain:

- multiple choice questions
- correct answers
- explanations

---

### Coding Challenge Generation

Claude can create coding challenges similar to LeetCode.

Each challenge must include:

- problem description
- difficulty level
- example input/output
- test cases
- solution explanation

---

### Learning Path Recommendations

Claude can analyze course progress and recommend learning paths.

Example:

HTML completed → recommend CSS → recommend JavaScript.

---

### Interview Preparation

Claude can generate technical interview questions for:

- Frontend
- Backend
- Algorithms
- System Design

---

## Developer Assistance

Claude should assist developers by:

- analyzing the repository
- identifying UI/UX improvements
- detecting bugs
- suggesting refactoring
- ensuring authentication and AI features work correctly

---

## Safety Rules

Claude must follow these rules:

- Do not break authentication logic
- Do not remove working APIs
- Do not rewrite the entire system
- Do not modify database schema without review

Claude should focus on **incremental improvements**.

---

## Learning Design Principles

All educational content generated should be:

- beginner friendly
- technically accurate
- clearly structured
- progressive in difficulty

Lessons should move from **beginner → intermediate → advanced**.

---

## Platform Goals

The goal of BlueLearnerHub is to create a **modern developer learning platform** combining the best ideas from:

- W3Schools
- LeetCode
- HackerRank
- Coursera

Claude should help improve the platform to deliver a high-quality learning experience for developers.
